目录结构介绍：
annotation  ：自定义注解相关
aspect      ：自定义切面配置
business    ：service 的业务聚合，减轻controller 的代码量
config      ：项目配置相关
contant     ：项目静态常量定义相关
controller  ：api 入口
dto         ：入参对象包
entity      ：数据库表映射包
exception   ：自定义异常
handler     ：项目自定义处理器
interceptor ：项目自定义拦截器
mapper      ：mybaties 相关接口
service     ：业务service
support     ：项目响应支持
util        ：项目工具包
vo          ：项目自定义响应对象
